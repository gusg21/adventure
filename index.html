ADVENTURE

An open-source system for making games in Batch. Meant for anyone to use.

A nice and easy way to make text-based adventure games! No coding needed! And using Batch! And it has advanced features for Batch developers! And it's open-source! My god, it's amazing!

Features:

  • Simple, easy-to-use interface (called your filesystem)

  • Infinitely huge game creation abilities (huge as in memes)

  • Advanced options for takin' it further (advanced as in easy-to-use)
